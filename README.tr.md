# 🛡️ Profil Rozetleri

Bu depo, GitHub üzerindeki tüm rozetlerin bir koleksiyonunu ve her birinin nasıl elde edileceğini içerir. Bunların bazıları alınabilir, bazıları alınamaz, bazıları ise halen beta test sürecindedir. Bu depoda rozetler `Çok Kolay`, `Kolay`, `Orta` ve son olarak `Zor` şeklinde sıralanmıştır.

GitHub rozetleri, ilk Mars helikopteri [Ingenuity](https://en.wikipedia.org/wiki/Ingenuity_(helicopter)) görevine başladıktan sonra eklendi.

> Günümüzde GitHub üzerinde toplam 11 rozet bulunmaktadır ve bazılarının bronz, gümüş ve altın seviyeleri de vardır.
